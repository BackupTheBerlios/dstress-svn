^ About

DStress is a collection of bug and language conformance test cases for the D programming language.
The unexpected output of a test case can have various reasons:

  1. a bug in the test case
  2. a failure of the build environment
  3. an inconsistent D-language specification or misunderstanding thereof
  4. a real compiler bug
  5. a bug in the system libs
Currently the dmd and gdc compilers are being tested.
